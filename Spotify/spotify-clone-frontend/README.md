# Spotify Clone Frontend

## Overview
This project is a frontend clone of the popular music streaming service, Spotify. It features a dark-themed user interface that is responsive and user-friendly. The application includes sections for navigation, playlists, a music player, and more.

## Features
- **Responsive Design**: The layout adapts to different screen sizes using CSS Flexbox and Grid.
- **Dark Theme**: The application uses a dark color palette with black/dark gray backgrounds, white text, and green accents.
- **Sticky Navigation**: The header remains fixed at the top while scrolling.
- **Sidebar Navigation**: A fixed vertical sidebar for easy access to different sections of the app.
- **Playlist and Album Cards**: Each album/playlist is displayed in a card format with hover effects.
- **Music Player**: A fixed bottom player with playback controls and a volume slider.

## Setup Instructions
1. Clone the repository to your local machine.
2. Navigate to the project directory.
3. Open `src/index.html` in your web browser to view the application.
